GitHub Proxy 加速


支持 GitHub API、Release、Archive 以及文件（除文件夹），右键复制出来的链接皆符合标准。

API、Release、Archive 使用 CF 加速，文件重定向至 JsDelivr

支持的链接格式如下：

分支源码：https://github.com/super-crayfish/project/archive/master.zip

Archive：https://github.com/super-crayfish/project/archive/v0.1.0.tar.gz

Release：https://github.com/super-crayfish/project/releases/download/v0.1.0/example.zip

分支文件：https://github.com/super-crayfish/project/blob/master/filename

Commit文件：https://github.com/super-crayfish/project/blob/111111/filename

Gist：https://gist.githubusercontent.com/super-crayfish/foo/raw/bar.py

API：https://api.github.com/repos/super-crayfish/project/releases/latest
